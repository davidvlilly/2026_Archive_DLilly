THIRD-PARTY LIBRARIES (via vcpkg, x64-windows-static)
================================================================================

  ImGui 1.91.9 (MIT License)
    Immediate-mode graphical user interface library. Provides windows,
    menus, buttons, tooltips, and text rendering. Used for all UI
    elements in the application. Includes Win32 and DirectX 11 backends
    for platform integration.

  ImPlot 0.16 (MIT License)
    Plotting library built on top of ImGui. Provides interactive 2D
    plots with zoom, pan, and mouse interaction. Used to display the
    signal waveform with colored segments and editing overlays.

  HDF5 1.14.6 — C++ API (BSD-3-Clause License)
    Hierarchical Data Format library for reading and writing structured
    scientific data files. Used via iFileIO to load and save H5A signal
    files containing time series, magnitude data, and state annotations.

  zlib 1.3.1 (zlib License)
    Compression library. Required by HDF5 for internal data compression
    of datasets within H5A files.

  libaec 1.1.3 (BSD-2-Clause License)
    Adaptive Entropy Coding library providing Szip compression. Required
    by HDF5 as an additional compression option for datasets.
